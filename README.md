# SketchFlow

Collects RSS feeds on a schedule, classifies them into ideas / issues / tools, and
generates KR/EN LLM summaries and pros/cons discussions.

Full documentation:
- 🇰🇷 [README.ko.md](README.ko.md)
- 🇺🇸 [README.en.md](README.en.md)

## Quick start

```bash
npm install
cp .env.example .env   # then edit as needed
npm start              # http://localhost:3000
npm test               # run the test suite
```
